# WMLS 空耳歌词互换卡片 — AI 出图指南

## 卡片尺寸
- **单卡尺寸：** 140 × 80mm（16:9）
- **含出血：** 146 × 86mm
- **AI 图片建议尺寸：** 800 × 450px，PNG 格式，透明背景效果最佳
- **图片位置：** 卡片左侧 42% 宽度区域

---

## Prompt 结构（通用模板）

```
[主体] + [五月天元素] + [氛围] + [配色] + [技术参数]

--ar 16:9 --style raw --no text, letters, words, Chinese characters
```

> 注意：Prompt 中**不要出现文字/歌词/歌名**，否则会被 AI 画成图案的一部分。

## 替换图片步骤

1. 用 AI 工具按上方 Prompt 出图
2. 下载 PNG 格式（推荐透明背景）
3. 打开对应的 `card-XX.html` 文件
4. 找到 `<!-- 替换方式：删除 .ai-placeholder div... -->` 注释区域
5. 删除 `.ai-placeholder` 整个 div
6. 取消下方注释，把 `{{AI_IMAGE}}` 替换为图片路径或 base64
7. 保存，浏览器打开预览打印

## 打印步骤

1. 打开 `front-BATCH.html` → 浏览器打印 → 另存为 PDF（正面）
2. 打开 `back-BATCH.html` → 浏览器打印 → 另存为 PDF（背面）
3. 把两份 PDF 合并（用 PDF 合并工具）
4. 发打印店：
   - 尺寸：每张 146×86mm（含出血）
   - 正面 PDF + 背面 PDF 对齐打印
   - **亮膜覆膜**
   - 拼版印刷（27张拼成大纸一次印，更便宜）
