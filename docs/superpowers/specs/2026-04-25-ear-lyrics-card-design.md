# WMLS 空耳歌词互换卡片设计

## 概念与目标

为五月天粉丝（WMLS）设计的空耳歌词互换实体物料卡片，16:9 横版，14×8cm 打印尺寸。
每张卡片正面展示一条五月天歌词的空耳版本，左侧配五月天元素图案，右侧放大字歌词，右下角附二维码供扫码查看正确答案并收听歌曲。

互换时对方猜歌，增加互动乐趣；扫码直接获得答案和歌曲链接，体验闭环。

---

## 设计规格

### 打印规格
- **卡片尺寸：** 140mm × 80mm
- **出血：** 四边各扩 3mm（最终尺寸 146mm × 86mm）
- **分辨率：** 300 DPI（对应像素约 1720 × 1014px）
- **色彩模式：** CMYK（标注在文件中，打印店会转换）
- **文件格式：** HTML 导出 PDF，或直接提供 PDF

### 分辨率换算
- 300 DPI 下：140mm = 1654px，80mm = 945px
- 保险起见按 1720 × 1000px 输出足够

## 视觉风格

### 整体风格
- 色彩：五月天应援色为主（蓝 + 粉荧光棒色），搭配胡萝卜橙
- 字体：中文手写体/综艺体，歌词突出放大
- 氛围：演唱会荧光棒感，暗色背景 + 发光文字

### 左侧图案区（AI生成）
以下元素供 Prompt 参考，具体每张卡可换不同成员/元素：

- 卡通成员头像（chibi 风格）
- 胡萝卜（wmls 自嘲符号）
- 荧光棒/应援灯牌
- 演唱会舞台/麦克风
- 成员 Q 版人物组合

### 右侧文字区
- 空耳歌词：超大字号，白色/荧光色，居右对齐
- 底部提示语：「扫码听歌 · 揭晓答案 →」
- 二维码：尺寸约 18×18mm，居右下角

---

## 二维码

- 链接指向：空耳测试结果页（`/ear/result`）或针对每张卡单独生成的歌曲页
- QR 码生成：可用草料二维码 API 或本地生成，确保扫描体验流畅
- 每张卡片对应独立 URL（包含歌曲 ID 参数）

---

## 技术方案

### HTML 打印模板
- 单文件 HTML + CSS，CSS 使用 `@media print` 控制打印样式
- 支持变量替换（歌词内容、二维码图片、左侧AI图）
- 导出方式：浏览器打印 → 另存为 PDF

### 输出流程
1. 用 AI 工具生成左侧图案（按 Prompt 出图）
2. 将 AI 图填入 HTML 模板对应位置
3. 填入空耳歌词文本
4. 填入/生成二维码图片
5. 浏览器打印为 PDF，发打印店

---

## 卡片内容清单（27道题）

每张卡片对应 `earQuestions.js` 中一道题目，包括：
- 空耳歌词（`earLyric`）
- 正确歌名（`correctAnswer.song`）
- 歌曲链接（二维码跳转目标）
