# 五月天空耳猜歌测试 - 设计方案

## 1. 项目概述

### 定位
独立于现有 WMLS 生活测试的空耳歌词趣味测试，10道题，每题即时反馈，兼具等级称号和详细解析双重结果。

### 目标
- 测试粉丝对五月天歌词空耳的敏感度
- 提供趣味分享点（五迷等级称号）
- 提供学习价值（详细解析了解原歌词）
- 通过结果页互相导流，提升两个测试的完测率

---

## 2. 核心决策

| 问题 | 决策 |
|------|------|
| 结果形式 | A+C：等级称号 + 详细解析 |
| 题目数量 | 10道 |
| 难度搭配 | B：中等难度混合，约5道常见+5道刁钻 |
| 作答反馈 | A：每题即时显示对错 |
| 入口关系 | A：独立入口（首页新增卡片） |
| 结果页导流 | 互相增加对方入口 |

---

## 3. 页面流程

```
首页 → [新增] 空耳测试入口卡片
  ↓
空耳答题页 → 展示空耳歌词 + 4个选项
  选完立即反馈：对/错 + 歌曲信息
  ↓

  [底部] → WMLS 测试入口
  ↓
WMLS 结果页 → [底部] → 空耳测试入口
```

## 4. 题目设计

### 题库来源
精选自提供的空耳素材，10道题。

### 题目结构
```javascript
{
  id: number,
  earLyric: string,      // 空耳歌词（如："难道我又我又住院了"）
  options: [
    { key: string, content: string, correct: boolean }
  ],
  correctAnswer: {        // 选完后的详细展示
    song: string,        // 歌曲名（"我又初恋了"）
    album: string,        // 专辑名（"为爱而生"）
    originalLyric: string // 原歌词（"难道我又我又初恋了"）
  }
}
```

### 干扰项策略
3个干扰选项取自同专辑或其他名曲，避免出现明显错误选项。

## 6. 技术实现

### 文件结构

```
backend/src/
  data/
    earQuestions.js          # 空耳题目数据（10道）
  routes/
    earQuestions.js           # GET /api/ear/questions
    earResults.js             # POST /api/ear/submit + GET /api/ear/result/:id

frontend/src/pages/
  HomePage.vue                # 新增空耳测试入口卡片
  EarQuizPage.vue             # 空耳答题页
  EarResultPage.vue           # 空耳结果页

frontend/src/router/index.js
  # 新增 /ear-quiz 和 /ear-result/:id 路由
```

### API 设计

| 接口 | 方法 | 描述 |
|------|------|------|
| `/api/ear/questions` | GET | 获取10道空耳题目 |
| `/api/ear/submit` | POST | 提交答案，获取结果ID |
| `/api/ear/result/:id` | GET | 获取结果详情（含称号+解析） |

### 结果数据结构

```javascript
{
  id: string,
  score: number,              // 答对题数（0-10）
  grade: {
    level: number,            // 1-6 级
    title: string,           // 称号
    desc: string              // 描述
  },
  questions: [                // 10题详细解析
    {
      earLyric: string,
      selected: number,       // 用户选的选项索引
      correct: boolean,
      correctAnswer: {
        song: string,
        album: string,
        originalLyric: string
      }
    }
  ]
}
```

## 7. UI 设计

### 空耳答题页

- 顶部：进度条 + 题号（如 "3/10"）
- 中部：大字展示空耳歌词（居中，有冲击力）
- 下部：4个选项按钮（类似 QuizPage.vue 风格）
- 选完后：弹出一个反馈卡片（延迟300ms出现），展示歌曲+专辑信息
- 反馈卡片消失后自动进入下一题

### 空耳结果页

- 顶部：等级称号大字展示 + 描述
- 中部：10题解析列表（可滚动）
- 底部：WMLS 测试导流卡片 + [重新测试] 按钮
